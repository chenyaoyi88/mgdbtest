## 做一个简单的小网站

无聊用 express + mongodb + angular+ 去做一个简单的小网站。

### 开发模式

前后端分离，后端服务提供接口，前端请求接口获取数据并实现路由和页面逻辑。

### 技术栈

后端：

- node      v9.1.0
- express   v4.14
- mongodb   v3.4

前端：

- angular+  v5.0.0

### 预期实现功能

1. 注册、登录、退出
2. 增删改查文章
3. 上传图片
4. 留言、评论
5. 用户权限

### 目前已完成功能

前端：
